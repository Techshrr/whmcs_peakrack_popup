# WHMCS PeakRack Popup 弹窗插件

这是一个适用于 WHMCS 9.x / PHP 8.3 的客户区弹窗插件，可用于优惠套餐、优惠码、域名优惠、线路维护、紧急公告和拼团活动。

## 下载和上传

插件完整上传包在：

```text
whmcs_peakrack_popup/
```

把该目录里的 `modules` 文件夹上传到 WHMCS 根目录即可：

```text
whmcs_peakrack_popup/modules -> 你的 WHMCS 根目录/modules
```

## 主要功能

- 后台弹窗新增、编辑、启用、停用、删除。
- 后台中文 / English 界面切换。
- 前台弹窗跟随 WHMCS 客户区语言显示中文或英文内容。
- 支持优惠套餐、优惠码、域名优惠、线路维护、紧急公告、拼团优惠、普通通知。
- 支持居中弹窗、顶部横幅、底部横幅、右下角浮窗。
- 支持每个弹窗独立选择强调色。
- 支持页面规则、受众规则、展示频率、开始/结束时间、延迟显示、自动关闭。
- 支持展示、点击、关闭基础统计。

## 安装

1. 上传 `whmcs_peakrack_popup/modules` 到 WHMCS 根目录。
2. 进入 WHMCS 后台 `系统设置 > 插件模块`。
3. 启用 `PeakRack Popup`。
4. 进入 `插件 > PeakRack Popup` 创建或编辑弹窗。

# WHMCS PeakRack Popup Addon

A bilingual WHMCS 9.x / PHP 8.3 client-area popup addon for promotions, coupon codes, domain deals, maintenance notices, urgent announcements, and group-buy campaigns.

## Package

The uploadable WHMCS package is in:

```text
whmcs_peakrack_popup/
```

Upload its `modules` folder to your WHMCS root:

```text
whmcs_peakrack_popup/modules -> your WHMCS root/modules
```
